# PhD Study Search Platform

## Overview
A web-based platform that allows users to search and explore PhD opportunities based on title, university, and subject.

## Features
- Search by keyword (title/university)
- Filter by subject
- Add new PhD opportunities
- Structured data storage using SQLite

## Technologies
- Python (Flask)
- SQLite
- HTML/CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
